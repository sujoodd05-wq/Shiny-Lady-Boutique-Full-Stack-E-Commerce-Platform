PROJECT NAME: SHINY LADY BOUTIQUE
=====================================================
Description: 
A luxury e-commerce platform for jewelry and accessories. 
The system features a premium Customer Interface and a 
comprehensive Admin Dashboard.

AUTHOR: Sujood Daghlas

=====================================================
1. SYSTEM REQUIREMENTS
=====================================================
- Local Server: XAMPP / WAMP / MAMP
- Database: MySQL
- Language: PHP 7.4+
- Icons: Bootstrap Icons (Included via CDN)

=====================================================
2. SETUP INSTRUCTIONS
=====================================================
1. Copy the 'web2026' folder into your 'C:/xampp/htdocs/' directory.
2. Start Apache and MySQL from the XAMPP Control Panel.
3. Open your browser and go to: http://localhost/phpmyadmin/
4. Create a NEW database named: boutique_db
5. Import the provided 'boutique_db.sql' file into the database.

=====================================================
3. ACCESS LINKS & CREDENTIALS
=====================================================
--- CUSTOMER INTERFACE ---
Link: http://localhost/web2026/customer/index.php
Features: Shop All pieces, Categories, User Registration, 
          Shopping Cart, and Beauty Compliment Banner.

--- ADMIN DASHBOARD ---
Link: http://localhost/web2026/admin/login.php
Credentials:
   - Username: admin
   - Password: password
Features: Stock Management, Order Tracking, Sales Analytics, 
          Activity Logs, and Member Control.

=====================================================
4. KEY FEATURES
=====================================================
- Responsive Design (Mobile & Desktop).
